健身健美分享会2019年11月09日

Written by Pepper Targaryen with ♥ on November 10, 2019 in 健身

交大第二届健体比赛开始了，本次有幸邀请了陈胄学长来分享他的经验。

陈胄学长目前是曾获得IFBB Elite Pro的证书，也是国家的健身健美教练，是美国品牌安菲特的代言人。

增肌减脂

力量训练

力量举（单项力量）、CrossFit（多种结合训练，为了综合素质）、大力士、专项运动、健美健身（理想体型）

抗阻力训练

自由重量 vs 固定器械

  • 自由重量建议安排75%动作
  • 自由重量可以找到自己的发力感觉
  • 固定器械主要给白领，没时间训练的人
  • 赛季的时候，由于脱水脱糖等原因实在没力气训练可以考虑机械
  • 多做杠铃和哑铃

复合动作 vs 孤立动作

复合动作主要是多关节，孤立动作主要是单关节

增肌和减脂的训练

  • 训练强度基本没什么差别
  • 做小重量会让身体觉得不需要那么多肌肉
  • 减脂期间的饮食要稍微调整
  • 增肌期间要多吃才有力气训练

肌肉维度和肌肉力量

  • 初期阶段肌肉维度越大肌肉力量越大
  • 后期技术不提高，力量上不去，容易出现肌肉代偿，借力
  • 规范动作之后，再上重量

有氧运动

  1. 慢速有氧：心率低（散步，遛狗）
  2. 中速有氧：120-140心率，可以小声说话，30-40分钟
  3. 高强度间歇性有氧（HIIT）：适合体重不是很大 增肌期间每周做1-2次有氧，减脂期间早上1次，训练后1次

健身饮食计划

营养素 能量 食品 建议用量
碳水 4千卡/克 大米、燕麦、水果 约3克/公斤
蛋白质 4千卡/克 鸡胸肉、牛肉、鱼肉、虾 减脂2.5-3克/公斤 增肌2-2.5克/公斤
脂肪 4千卡/克 油、坚果 减脂0.6-1克/公斤 增肌0.4-0.8克/公斤

摄入时间：

  • 早起：慢速碳水 + 蛋白质/脂肪
  • 运动前2-3小时：慢速碳水 + 蛋白质
  • 运动后：快速碳水（0.3-0.5克/千克）（运动饮料） + amino（3-5克）（20多种氨基酸）
  • 运动后1小时：淀粉类碳水 + 高质量蛋白质，可以吃一顿正餐 休息日可以吃的少一点，每天早上看一下自己的体重和腹部

营养品

  1. （推荐） 蛋白质、一水肌酸、amino、复合维生素片（早上）
  2. （还行） BCAA、氮泵、燃脂类补剂、关节宝
  3. （一般） 谷氨酰胺、左旋肉碱、鱼油、蛋白类（素食鸡胸肉等）
  4. （没用） 促睾类、ZMA、CLA、前体 蛋白质吸收速度：水解 > 分离 > 乳清 > 复合 = 矩阵 = 正氮 > 酪蛋白
