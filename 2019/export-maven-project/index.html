Export Maven Project with Dependencies

Written by Pepper Targaryen with ♥ on November 9, 2019 in 教程

We want to export a Java-based project to an executable jar file. The process to export a jar file is not the same with a Maven Project which contains external denpendencies.

The good news is that Maven provides a great plugin that is dedicated for creating executable JAR files with resources and dependencies. This plugin is called Maven Assembly Plugin.

To use the Maven Assembly Plugin, declare the following XML code in the <plugins> section which is under the <build> section of the pom.xml file:

<project ....>
    <modelVersion>1.0.0</modelVersion>
    <groupId>Your Group</groupId>
    <artifactId>Your Id</artifactId>
    <packaging>jar</packaging>
    <version>0.0.1-SNAPSHOT</version>
    <name>Your Name</name>
     
 
    <dependencies>
        ....
         
    </dependencies>
 
    <build>
        <plugins>
            <plugin>
                <artifactId>maven-assembly-plugin</artifactId>
                <configuration>
                    <archive>
                        <manifest>
                            <mainClass>The Class that Contains Main()</mainClass>
                        </manifest>
                    </archive>
                    <descriptorRefs>
                        <descriptorRef>jar-with-dependencies</descriptorRef>
                    </descriptorRefs>
                </configuration>
            </plugin>
        </plugins>
    </build>
</project>

One thing to remark is to include: <packaging>jar</packaging>

To generate the jar file, we can use command:

mvn clean install assembly:assembly

We can also build the project in Eclipse.

In Eclipse, right click on the project and select Run As > Maven build, and specify the goal assembly:assembly in the dialog, then click Run.

There would be two files in the target folder. One with all the dependencies which is much larger than the other.
