Written by Pepper Targaryen with ♥ on February 14, 2021 in 编程

工作中遇到的关于CSS的写法

单行省略

#div1{
    overflow: hidden;    
 	text-overflow: ellipsis;      //超出部分以省略号显示
 	white-space: nowrap;
 	width: 20em;  //用px单位亦可行
}

多行省略

#div1{
    overflow : hidden;
    text-overflow: ellipsis;
    display: -webkit-box;
    -webkit-line-clamp: 2;      /* 可以显示的行数，超出部分用...表示*/
    -webkit-box-orient: vertical;
}
